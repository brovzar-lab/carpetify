**Mock chain needed for documentStore.ts:**

The production code calls these Firestore methods in order:
1. `getFirestore()` -> `db`
2. `db.collection('projects').doc(projectId).collection('generated').doc(docId)` -> `docRef`
3. `docRef.get()` -> returns `{ exists, data() }` for the current document
4. `docRef.collection('versions')` -> `versionsRef`
5. `versionsRef.orderBy('version', 'asc').get()` -> returns `{ size, docs }` for version count check
6. `db.batch()` -> `batch`
7. `batch.delete(oldestDoc.ref)` (conditional, only when >= 10 versions)
8. `batch.set(versionsRef.doc(String(currentVersion)), versionDoc)` -> archives current
9. `batch.commit()` -> commits the batch
10. `docRef.set(docData)` -> writes new version
