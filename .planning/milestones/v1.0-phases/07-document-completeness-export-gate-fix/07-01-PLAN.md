<objective>
Align REQUIRED_DOCUMENTS namespace with DocumentChecklist tipo values so VALD-06 correctly identifies uploaded documents, and wire hasExclusiveContribution from financial data instead of hardcoded false.

Purpose: VALD-06 currently always reports uploaded documents as "missing" because its REQUIRED_DOCUMENTS keys (B1_producer, D1_seguro, C2a) don't match the tipo values stored by DocumentChecklist (cv_productor, cotizacion_seguro, indautor_guion). This is a production blocker that permanently blocks export for all projects. Additionally, hasExclusiveContribution is hardcoded false, meaning the E2 conditional document check never fires.

Output: Fixed validation constants, wired engine condition, updated tests -- all passing.
</objective>
