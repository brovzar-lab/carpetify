# Kaizen

Applies continuous improvement methodology with multiple analytical approaches, based on Japanese Kaizen philosophy and Lean methodology.

## When to Use This Skill

- Applies continuous improvement methodology with multiple analytical approaches, based on Japanese Kaizen philosophy and Lean methodology.

## Instructions

1. Identify the process or area to improve
2. Apply 5 Whys analysis to find root causes
3. Use PDCA (Plan-Do-Check-Act) cycle for improvements
4. Measure before/after with specific metrics
5. Document improvements for organizational learning
