# Claude Code Terminal Title

Gives each Claude Code terminal window a dynamic title that describes the work being done so you don't lose track of what window is doing what.

## When to Use This Skill

- Gives each Claude Code terminal window a dynamic title that describes the work being done so you don't lose track of what window is doing what.

## Instructions

1. Monitor the current task or command being executed
2. Generate a concise, descriptive title for the terminal window
3. Update the terminal title dynamically as work progresses
4. Use format: [project] task-description
