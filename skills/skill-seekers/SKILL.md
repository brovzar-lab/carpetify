# Skill Seekers

Automatically converts any documentation website into a Claude AI skill in minutes.

## When to Use This Skill

- Automatically converts any documentation website into a Claude AI skill in minutes.

## Instructions

1. Accept a documentation website URL
2. Crawl and extract structured content from the docs
3. Organize content into skill sections (instructions, examples, edge cases)
4. Generate a complete SKILL.md with proper YAML frontmatter
5. Include relevant code examples and API references
