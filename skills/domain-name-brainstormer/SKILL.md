# Domain Name Brainstormer

Generates creative domain name ideas and checks availability across multiple TLDs including .com, .io, .dev, and .ai extensions.

## When to Use This Skill

- Generates creative domain name ideas and checks availability across multiple TLDs including .com, .io, .dev, and .ai extensions.

## Instructions

1. Accept brand concept, keywords, and naming preferences
2. Generate 20+ creative domain name candidates
3. Check availability via WHOIS for .com, .io, .dev, .ai TLDs
4. Score names by memorability, length, and brand fit
5. Present ranked recommendations with availability status
