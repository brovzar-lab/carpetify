# File Deletion

Secure file deletion and data sanitization methods.

## When to Use This Skill

- Secure file deletion and data sanitization methods.

## Instructions

1. Identify files targeted for secure deletion
2. Apply appropriate sanitization method (overwrite, shred)
3. Verify deletion completeness
4. Log deletion actions for audit trail
5. Handle edge cases: open file handles, permissions, symlinks
