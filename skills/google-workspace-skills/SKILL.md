# Google Workspace Skills

Suite of Google Workspace integrations: Gmail, Calendar, Chat, Docs, Sheets, Slides, and Drive with cross-platform OAuth.

## When to Use This Skill

- Suite of Google Workspace integrations: Gmail, Calendar, Chat, Docs, Sheets, Slides, and Drive with cross-platform OAuth.

## Instructions

1. Authenticate via OAuth 2.0 with Google Workspace APIs
2. Gmail: send, search, label, draft emails
3. Calendar: create, update, query events
4. Sheets: read/write cells, formatting, formulas
5. Drive: upload, download, share, organize files
6. Slides: create, edit presentations programmatically
