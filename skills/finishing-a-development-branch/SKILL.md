# Finishing A Development Branch

Guides completion of development work by presenting clear options and handling chosen workflow.

## When to Use This Skill

- Guides completion of development work by presenting clear options and handling chosen workflow.

## Instructions

1. Check current branch status and uncommitted changes
2. Present options: merge to main, create PR, squash and merge, or rebase
3. Handle the chosen workflow end-to-end
4. Clean up the feature branch after merge if requested
