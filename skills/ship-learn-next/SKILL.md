# Ship Learn Next

Skill to help iterate on what to build or learn next, based on feedback loops.

## When to Use This Skill

- Skill to help iterate on what to build or learn next, based on feedback loops.

## Instructions

1. Review recent work outputs and feedback received
2. Identify patterns in what worked and what didn't
3. Suggest next actions based on highest-impact learning opportunities
4. Track iteration velocity and learning trajectory
5. Prioritize by impact × feasibility
