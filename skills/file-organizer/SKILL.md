# File Organizer

Intelligently organizes files and folders by understanding context, finding duplicates, and suggesting better organizational structures.

## When to Use This Skill

- Intelligently organizes files and folders by understanding context, finding duplicates, and suggesting better organizational structures.

## Instructions

1. Scan target directory and catalog all files
2. Detect duplicates by hash comparison
3. Categorize files by type, date, project, or content
4. Propose new organizational structure
5. Execute reorganization after user approval
