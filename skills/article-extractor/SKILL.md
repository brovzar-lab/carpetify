# Article Extractor

Extract full article text and metadata from web pages.

## When to Use This Skill

- Extract full article text and metadata from web pages.

## Instructions

1. Accept URL of article or web page
2. Extract main article content, stripping ads and navigation
3. Parse metadata: title, author, date, description, tags
4. Handle paywalled content gracefully (report limitation)
5. Return clean structured text with metadata
