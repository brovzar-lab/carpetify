# Review Implementing

Evaluate code implementation plans and align with specs.

## When to Use This Skill

- Evaluate code implementation plans and align with specs.

## Instructions

1. Compare implementation against specification document
2. Identify gaps, deviations, and missing requirements
3. Verify API contracts and data flow alignment
4. Generate compliance report with pass/fail per requirement
5. Suggest corrective actions for any misalignments
