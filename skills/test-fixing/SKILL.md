# Test Fixing

Detect failing tests and propose patches or fixes.

## When to Use This Skill

- Detect failing tests and propose patches or fixes.

## Instructions

1. Run test suite and collect failure reports
2. Analyze each failure: assertion error, timeout, setup issue
3. Propose targeted fix for each failing test
4. Distinguish between test bugs and implementation bugs
5. Apply fixes and verify all tests pass
