# Tapestry

Interlink and summarize related documents into knowledge networks.

## When to Use This Skill

- Interlink and summarize related documents into knowledge networks.

## Instructions

1. Scan a set of documents and extract key concepts
2. Identify relationships and cross-references between documents
3. Build a knowledge graph of connected concepts
4. Generate summaries at multiple levels of detail
5. Visualize the document network as an interlinked map
