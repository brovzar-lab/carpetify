# Markdown To Epub Converter

Converts markdown documents and chat summaries into professional EPUB ebook files.

## When to Use This Skill

- Converts markdown documents and chat summaries into professional EPUB ebook files.

## Instructions

1. Accept markdown files or chat transcript as input
2. Parse frontmatter for title, author, date metadata
3. Convert markdown to structured EPUB chapters
4. Apply professional typography and formatting
5. Generate table of contents automatically
6. Output .epub file to the specified destination
