# Raffle Winner Picker

Randomly selects winners from lists, spreadsheets, or Google Sheets for giveaways and contests with cryptographically secure randomness.

## When to Use This Skill

- Randomly selects winners from lists, spreadsheets, or Google Sheets for giveaways and contests with cryptographically secure randomness.

## Instructions

1. Load participant list from file, spreadsheet, or Google Sheet
2. Validate entries and remove duplicates if requested
3. Use cryptographically secure random selection (secrets module)
4. Select specified number of winners
5. Generate auditable selection report with timestamp and seed
