# Computer Forensics

Digital forensics analysis and investigation techniques.

## When to Use This Skill

- Digital forensics analysis and investigation techniques.

## Instructions

1. Collect and preserve digital evidence following chain of custody
2. Analyze file systems, logs, and memory dumps
3. Identify indicators of compromise (IOCs)
4. Timeline reconstruction from multiple evidence sources
5. Generate forensic investigation report
