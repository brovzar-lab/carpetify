# Software Architecture

Implements design patterns including Clean Architecture, SOLID principles, and comprehensive software design best practices.

## When to Use This Skill

- Implements design patterns including Clean Architecture, SOLID principles, and comprehensive software design best practices.

## Instructions

1. Apply SOLID principles to class and module design
2. Implement Clean Architecture layer separation
3. Use appropriate design patterns (Factory, Strategy, Observer, etc.)
4. Enforce dependency inversion and interface segregation
5. Review code for architectural violations and suggest improvements
