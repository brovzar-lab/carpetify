# Metadata Extraction

Extract and analyze file metadata for forensic purposes.

## When to Use This Skill

- Extract and analyze file metadata for forensic purposes.

## Instructions

1. Read EXIF, XMP, and filesystem metadata from files
2. Extract: creation date, author, GPS, software, modification history
3. Detect metadata anomalies or tampering indicators
4. Generate structured metadata report
5. Support common formats: images, PDFs, Office docs, videos
