# Csv Data Summarizer

Automatically analyzes CSV files and generates comprehensive insights with visualizations without requiring user prompts.

## When to Use This Skill

- Automatically analyzes CSV files and generates comprehensive insights with visualizations without requiring user prompts.

## Instructions

1. Load and parse CSV file, detecting column types automatically
2. Generate descriptive statistics (mean, median, std, distributions)
3. Identify correlations, outliers, and trends
4. Create visualizations (histograms, scatter plots, heatmaps)
5. Produce a structured summary report with key findings
